**Студент:** Симонов Илья Андреевич  
**Зачетная книжка:** 2023-01764  
**Группа:** 6311  

Репозиторий содержит цикл лабораторных работ, посвященных изучению и практическому применению технологий параллельных вычислений для решения вычислительно трудоемких задач (на примере перемножения квадратных матриц).

## Использованные технологии и инструменты
*   **Языки программирования:** C++ (стандарт C++11), Python (для генерации данных и верификации).
*   **Параллельные интерфейсы:** OpenMP, MPI (MS-MPI, Intel MPI), NVIDIA CUDA.
*   **Среда разработки:** VS Code, MinGW-w64 (WinLibs), CUDA Toolkit 13.2.
*   **Удаленные системы:** Суперкомпьютер СГАУ «Сергей Королёв» (планировщик Slurm).

## Аппаратные характеристики
*   **Локальный ПК:** Intel Core i7 (6 физических ядер / 12 логических потоков), NVIDIA RTX GPU.
*   **Кластер:** Узлы на базе процессоров Intel Xeon.

## Содержание работ

1.  **[Лабораторная работа №1: Последовательное умножение матриц](./LAB_1/README.md)**  
    Реализация базового алгоритма с кубической сложностью $O(N^3)$. Подготовка системы автоматической верификации результатов через NumPy.

2.  **[Лабораторная работа №2: Технология OpenMP](./LAB_2/README.md)**  
    Распараллеливание вычислений на многоядерном процессоре с общей памятью. Исследование масштабируемости и эффективности при разном количестве потоков.

3.  **[Лабораторная работа №3: Технология MPI](./LAB_3/README.md)**  
    Разработка программы для систем с распределенной памятью. Использование функций коллективного взаимодействия: `MPI_Scatter`, `MPI_Gather`, `MPI_Bcast`.

4.  **[Лабораторная работа №4: Технология NVIDIA CUDA](./LAB_4/README.md)**  
    Перенос вычислений на графический процессор (GPU). Сравнение "наивной" реализации, оптимизации через Shared Memory и профессиональной библиотеки cuBLAS.

5.  **[Лабораторная работа №5: Суперкомпьютер «Сергей Королёв»](./LAB_5/README.md)**  
    Освоение навыков работы с высокопроизводительными кластерными системами. Компиляция инструментами Intel и запуск задач через систему очередей Slurm.
