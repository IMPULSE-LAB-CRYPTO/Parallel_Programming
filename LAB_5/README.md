## 3. Порядок выполнения
1. Создана уникальная рабочая директория `Simonov_2023_01764_Lab5`.
2. Реализован код на C++ с использованием библиотеки MPI.
3. Для успешной компиляции в среде кластера была выполнена ручная настройка путей к инструментам Intel:
   ```bash
   source /soft/intel/parallel_studio_xe_2016.3.067/bin/psxevars.sh intel64
   ```
4. Программа скомпилирована с поддержкой стандарта C++11:
   ```bash
   mpiicpc -O3 -std=c++11 main.cpp -o matrix_mpi
   ```
5. Подготовлен скрипт запуска `submit_job.sh` для постановки задачи в очередь.

## 4. Скрипт для планировщика Slurm
```bash
#!/bin/bash
#SBATCH --job-name=Simonov_MPI_Lab5
#SBATCH --time=00:05:00
#SBATCH --nodes=2
#SBATCH --ntasks-per-node=4
#SBATCH --partition=batch
#SBATCH --output=result_simonov.out

# Активация среды Intel
source /soft/intel/parallel_studio_xe_2016.3.067/bin/psxevars.sh intel64

# Запуск программы на 8 процессах
mpirun ./matrix_mpi
```
